Introduction

무료 api를 사용하여 coin들의 정보를 보여주는 서비스입니다.

  • apexchart를 사용하여 코인의 가격을 CandleStick Chart의 형태로 시각화하였습니다.

  • styled component의 ThemeProvider를 사용하여 themeToggle를 구현하였습니다.

  • recoil을 사용하여 전역변수 관리를 해주었습니다.

  • data fetch는 react-query를 이용해 진행해주었습니다.

Quick Start

npm install
npm start
