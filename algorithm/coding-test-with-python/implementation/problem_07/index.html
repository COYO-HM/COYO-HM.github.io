[이코테] 구현 문제 7 : 럭키 스트레이트
2022.10.19.
Problem / Implementation
이 글은 "이것이 코딩 테스트이다."를 읽고 공부한 기록입니다.

내 풀이

N = input()
length = len(N)

front = sum([int(i) for i in list(N[0: length // 2])])
back = sum([int(i) for i in list(N[length // 2:])])

result = "LUCKY" if front == back else "READY"

print(result)
